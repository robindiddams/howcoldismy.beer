How it works

  1. Claim your username below — you'll get a token. Save it.
  2. Send your beer's temperature to the API with your token.
  3. Share your link: howcoldismy.beer/t/yourname
  4. Your temperature page auto-refreshes so everyone knows the cold truth.

API Reference

POST /api/claim

Body: {"username": "yourname"}

Returns: {"username", "token"}

POST /api/temperature

Body: {"token": "...", "temperature": 38, "unit": "F"}

GET /api/temperature/<username>

Returns the current temperature for a user.

GET /t/<username>

Human-readable temperature page. Auto-refreshes.

Tokens expire if unused for 8 days. Claim again if that happens.
